## 解题思路 
- 位运算
  - 使用位运算或者电路里面的异或门的性质
    - N XOR N = 0
    - 0 XOR N = N
  - 我们用0来异或所有的数
  - 如果数字出现两次会抵消
  - 所以最终的结果就是仅出现一次的数
  - 时间复杂度$O(n)$
  - 空间复杂度$O(1)$
  
## 代码
``` cpp
/*
 * @lc app=leetcode.cn id=136 lang=cpp
 *
 * [136] 只出现一次的数字
 */

// @lc code=start
#include<vector>

using namespace std;

class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int num = 0;
        // as all other number appears twice
        // 0 xor n xor x = 0
        // 0 xor n = n
        for (int i = 0; i < nums.size(); i++) num ^= nums[i];
        return num;
    }
};
// @lc code=end


```
